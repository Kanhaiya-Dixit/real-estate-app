# 🏡 Real Estate & Real-Time Chat Web Application

A full-stack modern real estate web application built using the MERN stack (MongoDB, Express.js, React.js, Node.js), Prisma ORM, and Socket.io for real-time buyer-owner communication.

---

## 🚀 Key Features

* **Advanced Property Search & Filters:** Users can search properties by location, price range, and property type with real-time database queries.
* **Interactive Map Integration:** Displays property locations visually using map components for better user navigation.
* **Secure Authentication & Authorization:** Implements JSON Web Tokens (JWT) and HTTP-only cookies to secure user sessions and protect routes.
* **Real-Time Chat System:** Powered by **Socket.io**, enabling instant messaging between buyers and property owners with unread notification badges and live updates.
* **User Profile & Post Management:** Authenticated users can update their profiles, upload avatars, and create, manage, or delete their property listings.
* **Responsive UI/UX:** Designed with a clean, modern interface optimized for various screen sizes.

---

## 🛠️ Tech Stack

### **Frontend:**
* **React.js (Vite):** For building a fast and dynamic user interface.
* **Tailwind CSS / SCSS:** For responsive styling and modern design.
* **Zustand / Context API:** For state management.

### **Backend:**
* **Node.js & Express.js:** RESTful API architecture and server-side logic.
* **Socket.io:** Real-time bi-directional event-based communication.

### **Database & ORM:**
* **MongoDB:** NoSQL database for flexible data storage.
* **Prisma ORM:** Modern database toolkit for type-safe database operations and modeling.
